
1. U cmd se navigirati u folder u kome zelite da se napravi aplikacija
2. U cmd napraviti novu angular aplikaciju sa ng new <ime_aplikacije>
3. Kopirati src i backend foldere, kao i fajlove server.js i package.json u folder aplikacije
4. U cmd izvrsiti komandu "ng add @angular/material"
5. U cmd izvrsiti komandu "npm install --save express"
6. U cmd izvrsiti komandu "npm install --save-dev nodemon"
7. U cmd izvrsiti komandu "npm install --save body-parser"
8. U cmd izvrsiti komandu "npm install --save mongoose"
9. Pokrenuti server sa "npm run start:server"
10. Pokrenuti aplikaciju sa "ng serve"
